#1 Infeasibility penalty is scale-dependent across expiries High
sdevpy/volatility/localvol/localvol_calib.py : 189

When LvObjectiveBuilder.check_params() returns infeasible, the objective returns self.cf_prices.sum(). This sum depends on the number of strikes and absolute price levels at the current expiry — so as the outer calibration loop advances through expiries, the penalty magnitude shifts. Equally infeasible parameters produce different penalty values at different expiries, giving the optimizer an inconsistent cost landscape. TsIvObjectiveBuilder already uses FLOAT_INFTY consistently; the open question in the comment should be resolved. 

-            return self.cf_prices.sum()
+            return constants.FLOAT_INFTY

#9 implied_vols() return type is inconsistent in the scalar branch Low
sdevpy/analytics/black.py : 41–42

implied_vols() is annotated -> npt.NDArray[np.float64], but when both strike and fwd_price are scalar it returns the bare float from implied_vol() without wrapping it in an array. Callers that unconditionally call array methods (.shape, indexing, np.allclose) on the result will fail silently for scalar inputs. 

-        return implied_vol(expiry, strike, is_call, fwd, fwd_price)
+        return np.float64(implied_vol(expiry, strike, is_call, fwd, fwd_price))

Alternatively, update the annotation to -> float | npt.NDArray[np.float64] if the scalar path is intentional.

#10 formula parameter has no type annotation in ParamLocalVolSection Low
sdevpy/volatility/localvol/localvol.py : 43

ParamLocalVolSection.__init__(self, time: float, formula) — the formula callable is the central dependency of every local-vol section (it computes LV at each PDE step) but carries no type hint. Every subclass constructor inherits this gap. Static type-checkers cannot verify that subclasses pass a compatible callable, and readers must trace call sites to infer the expected signature (t: float, x: ArrayLike, params: ArrayLike) -> ArrayLike. 

from typing import Callable
-    def __init__(self, time: float, formula):
+    def __init__(self, time: float,
+                 formula: Callable[[float, npt.ArrayLike, npt.ArrayLike], npt.ArrayLike]):
